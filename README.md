# slop-and-chop
Letting AI make up recipes

## Folder Structure

All recipes live inside the [`recipes/`](recipes/) directory and are organised by food type:

```
recipes/
├── appetizers/                  # Starters, snacks, and small bites
├── soups-and-stews/             # Broths, chowders, bisques, and hearty stews
├── salads/                      # Green salads, grain salads, composed salads
├── main-courses/                # Entrées by protein or dietary preference
│   ├── beef/
│   ├── chicken/
│   ├── pork/
│   ├── seafood/
│   ├── vegetarian/
│   └── pasta/
├── side-dishes/                 # Vegetables, grains, and accompaniments
├── breads-and-baked-goods/      # Loaves, rolls, muffins, flatbreads
├── desserts/                    # Cakes, cookies, pastries, frozen treats
├── drinks-and-beverages/        # Cocktails, mocktails, smoothies, hot drinks
└── sauces-and-condiments/       # Sauces, dressings, dips, and spreads
```

Each recipe is a Markdown file named after the dish (e.g., `smoky-ember-chili.md`). See [`recipes/README.md`](recipes/README.md) for the full recipe template.
